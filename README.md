# Generate 10k PFP from layers

This project is a simple script to generate 10k PFP from layers in Python.

## Run

```bash
pipenv shell
pipenv install --ignore-pipfile
python3 generate_pfp.py
```

## Output

The outputs are stored into `output` folder.
